# 💇 HairGo

A modern mobile application connecting clients with professional hairstylists for home salon services.

Built using React Native (Expo) and Firebase, HairGo enables users to discover stylists, request quotes, manage bookings, chat in real time, and showcase professional portfolios.

---

## ✨ Features

- 👤 Client & Stylist Authentication
- 📍 Location-based bookings
- 💬 Real-time chat
- 📸 Stylist portfolio
- 📅 Booking management
- 💰 Quote generation
- ⭐ Ratings & Reviews (planned)
- 🔔 Notifications (planned)

---

## 🛠 Tech Stack
